FAQ · 常見問題
Token 是怎麼計算的？ / How are tokens counted?

本工具用 GPT 系列分詞器估算 token 數，不同模型實際分詞略有差異，數字僅供參考。Tokens are estimated using a GPT-style tokenizer; actual counts vary slightly by model.

這個工具免費嗎？ / Is it free?

完全免費，全部在你瀏覽器本機運算，不會上傳任何文字。Completely free and runs locally in your browser; your text is never uploaded.

顯示的 API 價格準確嗎？ / Are the API prices accurate?

價格為各模型每百萬 token 的參考單價，可能變動，請以官方公告為準。Prices are reference rates per 1M tokens and may change; always check official pricing.
